Dokku Installation

Instructions to install dokku for Node.js and MongoDB in an Ubuntu instance

View the Project on GitHub marcochavezf/dokku-installation

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

  • Create a droplet with Ubuntu

  • Install dokku (http://dokku.viewdocs.io/dokku/installation/):

installs dokku via apt-get

wget https://raw.githubusercontent.com/dokku/dokku/v0.4.13/bootstrap.sh

sudo DOKKU_TAG=v0.4.13 bash bootstrap.sh

  • Config dokkut vía web browser (it’s recomendable to point dns to IP droplet).

  • Create the app (http://dokku.viewdocs.io/dokku/application-deployment/):

dokku apps:create ruby-rails-sample

  • Install mongodb plugin:

"install the postgres plugin"

"plugin installation requires root, hence the user change"

dokku plugin:install https://github.com/dokku/dokku-mongo.git mongo

"create a postgres service with the name rails-database"

dokku mongo:create rails-database

  • Link the database created with the app:

"each official datastore offers a link method to link a service to any application"

dokku mongo:link rails-database ruby-rails-sample

  • Deploy the app:

git remote add dokku

dokku@midomain.tld:ruby-rails-sample

git push dokku master
